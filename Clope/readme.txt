 Реализация алгоритма Clope.

 1. Точка старта - функция main() в файле Clope.cpp

 2. В командной строке могут быть переданы следующие аргументы:
        r:<коэффициент отталкивания>
        n:<количество объектов>
        i:<максимальное количество итераций>
        p:<путь к файлу данных>
    Значения по умолчанию:
        r = 2.7;
        n = 126;
        i = 1;
        p = "d:\\agaricus-lepiota.data";

3. По окончании работы в текущей папке записываются два файла с результатами:
        results.csv - в каждой строке номер кластера, количество транзакций, список номеров транзакций
        resultsEd.csv - то же, но вместо номера транзакции символ 'e' (съедобный) или 'p' (несъедобный)
    Пустые кластеры пропускаются.
